<h2> Como usar a ferrenta?</h2>
Para rodar a ferramenta é necessário instalar o Node em seu PC.
Depois de instalado vamos para os comandos...

1° Comando:

```bash
  npm install
```
2° Comando: 
```bash
  node goatmail.js
```
